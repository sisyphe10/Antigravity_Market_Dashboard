## 구조 스펙

- **상단 네비**: 배경 `#101418`, 하단 2px 앰버 라인, 브랜드 `AGE OF EMERGENCE`(800, letter-spacing 3.5px, 흰색), 활성 탭 = 앰버 배경 + `#101418` 글자. 높이 54px.
- **서브탭 스트립**: 네비 바로 아래 밀착(sticky top:54px), 풀블리드, 배경 `#161b21`, 하단 1px `#2a323b`, 탭 높이 38px·직사각(radius 0)·0.85rem/600, **가운데 정렬**, 활성 = 흰 글자 700 + 앰버 2px 밑줄. 알약(pill) 탭 금지.
- **페이지 큰 제목 금지**: 위치는 스트립이 말해준다. Updated 표기만 우상단(0.78rem, 이탤릭, `#8a919a`).
- **차트 패널은 흰색 유지**: Chart.js 텍스트가 다크 전제. 차트를 담는 컨테이너만 흰색, 목록/선택 UI는 다크.
- **버튼 위계(앰버 위계형)**: 주요 액션=앰버 채움(+ 글자), 보조=앰버 고스트(투명+앰버 보더), 중립=회색 고스트( 보더). 빨강/파랑 채움 버튼 금지.
- 날짜 입력: 크로미움 ko 로캘의 `type=date`는 끝에 하이픈이 붙음 → 표시가 중요한 곳은 `type=text`(YYYY-MM-DD).

## 신규 페이지 적용 방법

1. **스냅숏 서빙 페이지**(publish_snapshot 경유): 자동. `scripts/compose_personal_view.py`의
   `<style id="aoe-terminal-dark">`가 스냅숏의 모든 `*.html`에 주입된다. 페이지 고유 클래스가
   어색하게 보이면 그 블록에 보정 규칙을 추가한다(생성기 원천 수정 불요).
2. **별도 서빙 앱**(Caddy reverse_proxy: watchlist=quoteboard/, wiki=datalake/webui/static/ 등):
   주입이 닿지 않으므로 소스에 위 팔레트를 직접 적용한다.

## 함정 (실측 이력)

- `!important`끼리는 명시도 승부: 일반 `td:not(...)`(0,1,1)이 `.pos/.neg`(0,1,0)를 덮는다 → 등락색은 `td.pos` 병기.
- JS가 `el.style`을 만지면 인라인 속성이 재직렬화된다(`#000`→`rgb(0, 0, 0)`) → `[style*="..."]` 문자열 매칭 셀렉터는 JS가 건드리는 요소에 금물.
- `#fff` 일괄 치환 금지: 텍스트용 흰색(브랜드·호버·버튼 글자)까지 카드색으로 바뀐다.
- 인라인 밝은 배경 셀(히트 틴트 등) 위 글자는 어둡게 유지해야 읽힌다.
